    JazzPunkyy é o projeto de um buscador descentralizado e de código aberto. Nosso objetivo é construir uma plataforma onde você pode procurar sites sem intermediários usando seu próprio computador para buscar e armazenar dados. Opcionalmente, você pode participar da nossa rede para lucrar com o trabalho realizado pelo seu computador.

    Código-Fonte Guia


  • financiando o projeto

    • Você pode obter rendimentos adquirindo o nome exclusivo de uma página e toda vez que alguém pesquisar por ela você receberá uma quantia.


  • contribuindo com código e novas funcionalidades

    • Você pode expandir o projeto usando a sua linguagem favorita e arquitetura de microsserviços para ter acesso a potenciais ganhos futuros.
